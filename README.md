# ZenTide (禅潮)

🧘 专注类 iOS 应用 — 类似潮汐 App 的沉浸式专注计时器

## 功能

- 🎯 番茄钟专注计时（15/25/30/45/60 分钟可选）
- 🌊 4 种自然白噪声场景（海浪、雨声、森林、篝火）
- ✨ 动态粒子效果 + 呼吸光晕动画
- 📊 专注统计（今日/本周/累计）
- 🌙 全深色沉浸式设计

## 技术栈

- Swift + SwiftUI
- MVVM 架构
- AVFoundation 音频
- Canvas 粒子渲染
- iOS 17.0+

## 开发

1. 在 Xcode 中打开 `ZenTide.xcodeproj`
2. 选择 iPhone 模拟器
3. Cmd+R 运行
